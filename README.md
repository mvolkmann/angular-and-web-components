# angular-and-web-components

This is a web app that demonstrates
using vanilla web components in an Angular app.

To run this:

1. `npm install`
1. `npm start`
1. Browse localhost:4200.
1. Click the radio buttons and note how
   the color of the text above them changes.

The main source files to examine are all in the `src/app` directory:

- `hello-world.js` defines a web component
- `radio-group.js` defines a web component
- `app.html` uses the two web components
- `app.ts` implements event handling
